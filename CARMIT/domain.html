לחבר דומיין לאתר

שני חלקים פשוטים: כרמית קונה את הדומיין (הכתובת), ואבי מחבר אותו לאתר טכנית. כל אחד רואה כאן בדיוק מה עליו לעשות.

כרמית

לקנות את הדומיין

~5 דקות
1

בחרי שם

  • שם קצר, קל לזכור וקל לאיית - עדיף בלי מקפים ומספרים.
  • סיומת .com (מומלץ) או .co.il.
2

קני באתר Name.com

דרך www.name.com - זה המקום שאבי עובד איתו, כך החיבור יהיה מהיר וחלק.

  1. נכנסים ל-name.com, מקלידים בחיפוש את השם + הסיומת.
  2. אם כתוב "Available" - לוחצים "Add to cart".
  3. Checkout ← נרשמים (אימייל+סיסמה) ← משלמים בכרטיס.

שמרי היטב את פרטי ההתחברות ל-Name.com - צריך אותם לחידוש שנתי.

3

רק הדומיין - כלום נוסף

ינסו למכור לך Hosting / Email / Website Builder - דלגי על הכול. Whois Privacy אם בחינם - אפשר להשאיר. כל דבר בתשלום - להסיר.

✦ ואז - שלחי לאבי:
  • הדומיין המדויק שקנית.
  • גישה לחשבון Name.com (אימייל+סיסמה, או הוסיפי אותו כמשתמש). את לא עושה כלום נוסף.
אבי

החיבור הטכני

רשם → Cloudflare → Vercel
1

הכנה

  • ודא שהאתר של כרמית פרוס כפרויקט Vercel נפרד בשורש (לא כתת-נתיב), כדי שהדומיין יצביע ישר אליו. אם הוא כרגע תת-נתיב - שכפל אותו לפרויקט משלו או הגדר rewrite.
  • קבל מכרמית גישה לרשם (Name.com) ואת שם הדומיין.
2

Cloudflare - הוסף את הדומיין

  • ב-Cloudflare: Add a site ← הכנס את הדומיין (תוכנית Free).
  • Cloudflare ייתן 2 nameservers (כמו irena/ishaan.ns.cloudflare.com).
  • ב-Name.com ← Nameservers ← החלף לאלה של Cloudflare. (זהה לסטאפ של avisx-director.com.)
3

Vercel - הוסף דומיין

  • בפרויקט של כרמית: Settings → Domains → Add ← הכנס את הדומיין (וגם www).
  • Vercel יציג את הרשומות הדרושות.
4

Cloudflare - רשומות ה-DNS

  • Apex (דומיין נקי): A record ← 76.76.21.21
  • www: CNAME ← cname.vercel-dns.com
  • חשוב: Proxy = DNS only (ענן אפור), לא Proxied - אחרת ה-SSL של Vercel מתנגש.

חלופה: להצביע nameservers ישר ל-Vercel (ns1/ns2.vercel-dns.com) ולתת ל-Vercel לנהל DNS - אבל עדיף להישאר עם Cloudflare כמו בשאר האתרים.

5

אימות ו-SSL

  • חכה להתפשטות DNS (דקות עד שעות). Vercel יסמן ✓ Valid וינפיק SSL אוטומטית.
  • בדוק שהאתר עולה על הדומיין + על www, וש-http עובר ל-https.

זהו - האתר חי על הדומיין.
