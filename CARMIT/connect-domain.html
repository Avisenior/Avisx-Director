לחבר את הכתובת לאתר

הדומיין כבר קנוי ויושב ב-Cloudflare — נשאר רק להוסיף 2 שורות שמפנות אותו לאתר. 3 דקות עבודה. הכל קורה בתוך Cloudflare.

1

נכנסים להגדרות ה-DNS

  • ב-Cloudflare, בוחרים את הדומיין carmitskajo.com.
  • בתפריט הצד: DNS ← Records (רשומות).
2

מוסיפים רשומה ראשונה (A)

לוחצים Add record וממלאים בדיוק ככה:

Type
A
Name
@
IPv4 address
76.76.21.21
Proxy status
DNS only (ענן אפור ⬜, לא כתום)

לוחצים Save.

3

מוסיפים רשומה שנייה (CNAME)

שוב Add record:

Type
CNAME
Name
www
Target
cname.vercel-dns.com
Proxy status
DNS only (ענן אפור ⬜)

לוחצים Save.

!

הכי חשוב: הענן חייב להיות אפור

ליד כל רשומה יש אייקון ענן. אם הוא כתום (Proxied) — לחצו עליו כדי שיהפוך לאפור (DNS only). ענן כתום שובר את החיבור ל-Vercel וגורם לשגיאת אבטחה. אפור = תקין.

4

זהו — מחכים

  • תוך כמה דקות (לפעמים עד שעה) Vercel יזהה, יסמן ✓, ויפעיל אוטומטית תעודת אבטחה (HTTPS).
  • בודקים: נכנסים ל-carmitskajo.com — האתר של כרמית אמור לעלות.

(אם ב-Vercel הדומיין עדיין לא נוסף: פרויקט carmit-site ← Settings ← Domains ← Add ← carmitskajo.com.)
